# 📊 SQL Advanced Portfolio Project

## 📌 Overview

End-to-end SQL portfolio showcasing data analysis, window functions, joins, views, and financial calculations across real-world domains like **banking, stock market, retail, and sports (IPL)**.

This project demonstrates practical SQL skills used in real-world analytics and business scenarios.

---

## 🚀 Key Features

* ✅ Window Functions (RANK, DENSE_RANK, ROW_NUMBER, LAG, LEAD)
* ✅ Running Totals & Moving Averages
* ✅ Joins & Relational Data Modeling
* ✅ Views for Data Security
* ✅ User Defined Functions (EMI Calculation)
* ✅ Multi-domain datasets (Banking, Retail, IPL, Stock Market)

---
